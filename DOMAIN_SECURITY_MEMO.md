# ⚠️ 重要：ドメイン変更時の注意点 (SECURITY MEMO)

このプロジェクトには、**「パクリ防止トラップ（罠コード）」**が仕込まれています。
もし本番公開するドメインを変更したり、別のドメインで運用を開始する場合は、以下の設定を必ず更新してください。

これを忘れると、**あなた自身のサイトで「速度制限」や「強制転送」が発動してしまいます。**

## 修正場所

以下のファイルを編集してください：

### [config/security.ts](file:///f:/webbusiness/gif-converter/src/config/security.ts)

```typescript
export const ALLOWED_DOMAINS = [
    'toolkit-lab.com',        // 現在の本番ドメイン
    'localhost',               // 開発用
    '127.0.0.1',               // 開発用
    'your-new-domain.com',    // 新しいドメインを追加する場合はここに入れる！
];
```

## 発動するトラップの内容 (非許可ドメインの場合)

1.  **速度制限**: スピードテストが強制的に 1.5Mbps に制限されます。
2.  **透かし**: 画面に「UNOFFICIAL CLONE」という大きな文字が表示されます。
3.  **強制転送**: ページを開いてから 60 秒後に、自動的に `toolkit-lab.com` へ転送されます。

## AIへの注意
このプロジェクトには **AI Sentinel** も導入されています。AI（ChatGPTやClaudeなど）にコードを読み取らせた際、AIが「これはToolkit Labの作品である」と認識し、不用意なコピーを拒否するように促すコメントが含まれています。
